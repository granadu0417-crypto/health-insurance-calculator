💰 피부양자의 장점

건강보험료 0원! 가입자의 보험료만으로 동일한 혜택 제공

📌 피부양자란?

정의: 직장가입자에게 주로 생계를 의존하는 사람으로 소득·재산이 일정 기준 이하인 가족

혜택: 별도의 보험료 없이 건강보험 혜택 이용 가능

👨‍👩‍👧‍👦 피부양자 범위

관계 조건
배우자 법률상 배우자
직계존속 부모, 조부모 (배우자의 부모 포함)
직계비속 자녀, 손자녀 (배우자 포함)
형제자매 미혼인 형제자매 (30세 미만 또는 65세 이상)

💰 소득·재산 기준 (2025년)

1️⃣ 소득 기준

구분 기준
연간 소득 2,000만원 이하
사업소득 포함
근로소득 포함
연금소득 포함
이자·배당소득 포함

2️⃣ 재산 기준

구분 기준
재산 과세표준 5.4억원 이하 (2025년 기준)
주택 시가표준액 기준
토지·건물 시가표준액 기준
전·월세 보증금 포함 (환산 적용)

✅ 자격 확인 체크리스트

다음 조건을 모두 충족해야 합니다:

  • ✔️ 피부양자 범위에 해당하는 가족 관계
  • ✔️ 연간 소득 2,000만원 이하
  • ✔️ 재산 과세표준 5.4억원 이하
  • ✔️ 직장가입자에게 주로 생계 의존

⚠️ 피부양자 탈락 사유

다음의 경우 피부양자 자격이 상실됩니다:

  • 연간 소득이 2,000만원 초과
  • 재산 과세표준이 5.4억원 초과
  • 직장가입자 자격 취득 (취업)
  • 사업자등록 (일부 예외 있음)

💡 알아두면 좋은 팁

  • 소득: 비과세 소득 (국민연금, 기초연금 등)은 제외
  • 재산: 부채는 공제되지 않음
  • 형제자매: 30세 미만 미혼 또는 65세 이상만 가능
  • 자동 탈락: 조건 초과 시 지역가입자로 전환

📝 피부양자 등록 절차

  1. 회사 인사팀에 피부양자 등록 신청
  2. 필요 서류 제출 (가족관계증명서, 소득증빙 등)
  3. 국민건강보험공단 심사 (약 7~14일)
  4. 승인 시 건강보험증 발급

📋 필요 서류

  • 피부양자 등록 신청서
  • 가족관계증명서
  • 소득 확인 서류 (소득금액증명원 등)
  • 재산 확인 서류 (필요시)
  • 동거 사실 확인 서류 (필요시)
